# FastFurnace

A Minecraft Paper plugin that allows you to speed up furnace smelting across the entire server.

## Features
- `/fastfurnace on`: Enables the speed boost.
- `/fastfurnace off`: Disables the speed boost.
- `/fastfurnace <number>`: Sets the smelting speed multiplier (1-1000).

## Build
This project uses Maven. Run `mvn package` to build the JAR file.
